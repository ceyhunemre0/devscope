# devscope

Self-hosted AI-powered developer productivity engine. Scans your local git repos and produces standup-style summaries.

## Status

Pre-alpha. See `docs/superpowers/specs/` (local-only) for design.

## Quickstart

```bash
pipx install -e .
devscope init
devscope projects add ~/path/to/repo
devscope today
```
